# PiEEG Software

PiEEG Server turns supported PiEEG hardware into a real-time EEG streaming platform with a browser dashboard, WebSocket API, CSV recording, Lab Streaming Layer output, VRChat OSC, and automation hooks.

For browser apps that talk to a board directly (no server), see the
[JavaScript SDK](/software/api/javascript-sdk) — Web Bluetooth / Web Serial with
built-in filtering and band-power analysis.
