# JNEEG

JNEEG
JNEEG is shield for the NVIDIA Jetson Nano that measures biosignals — including EEG (electroencephalography), EMG (electromyography), and ECG (electrocardiography) — across 8 simultaneous channels.
What It Does
JNEEG brings the PiEEG biosignal acquisition experience to the NVIDIA Jetson Nano platform. It connects directly to the Jetson Nano GPIO header and uses the ADS1299 analog front-end chip to capture 8 channels of high-quality biosignal data in real time.
Because it runs on the Jetson Nano, JNEEG is particularly well suited for applications that benefit from on-device GPU acceleration — such as real-time machine learning inference on EEG data, neural decoding pipelines, and computer vision combined with BCI input.

Key Features:
- 8 channels of simultaneous biosignal acquisition
- Supports EEG, EMG, and ECG measurements
- Connects directly to the NVIDIA Jetson Nano GPIO header
- Powered by the ADS1299 analog front-end chip
- Compatible with dry and Ag/AgCl wet electrodes via 2.54 mm pin pitch connectors
- On-device GPU acceleration available via Jetson Nano
