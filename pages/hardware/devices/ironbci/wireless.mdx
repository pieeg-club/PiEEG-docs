# Software
IronBCI streams biosignal data wirelessly over BLE. There are three ways to receive and work with the data — choose the one that best fits your use case.

### Option 1 — PiEEG Server Integration

Official PiEEG-Server see on the Doc - Software. Also available for ironbci in GitHub via the following [link](https://github.com/pieeg-club/PiEEG-server) 

![Image](/images/hardware/2025/05/pieeg-1024x485.jpg)



**Easy start. To measure data, just lauch app or Python Script. **

We provide an SDK for an** open-source** mobile App (Android). Mobile app allows to see data in real-time, perform signal processing for real-time data, and save data in Excel format.

[link is here](https://github.com/pieeg-club/ironbci/tree/master/mobile_app)

![mobile application for brain-computer interface](/images/hardware/2024/11/app-300x252.png)

We provide an open-source Python SDK for saving data, signal processing, and real-time visualisation. [link is here](https://github.com/pieeg-club/ironbci/tree/master/SDK)

![Image](/images/hardware/2025/07/image_3-300x28.png)

If you would like change protocol for data transfer or adjust registers you can use Framework for STM32 for data transfer in C via STM32CubeMX.
