# How to start. Software with ardEEG

Scripts available in GitHub. Read data with a band-pass filter and real-time visualisation with Arduino for graph Arduino [script](https://github.com/Ildaron/ardEEG/blob/main/software/arduino/1.Send_data_via_wifi.ino), please upload via Arduino IDLE and just only write your Wi-Fi and password

```
`char ssid[] = "Wifi Name"; char pass[] = "password";`
```

and a Python [script](https://github.com/Ildaron/ardEEG/blob/main/software/python/1.Alpha_real_time.py), for Graph visualisation

For power supply, use only a battery with 5V for JetsonNano and for the monitor, as in the picture as in the photo above.
