### SDK

## Scripts available in [GitHub](https://github.com/pieeg-club) Read data with band-pass filter and real-time visualisation with[RaspberryPi5](https://github.com/pieeg-club/PiEEG-16/blob/main/GUI/Graph_Gpio_D%20_1_5_4_not_spike.py)

To save Raw data to excel you can use the next [script for Pi5](https://github.com/pieeg-club/PiEEG-16/blob/main/Save_data/1.Save_Data.py)

**Libraries** for Python 3.11. 2

- import spidev (pip install spidev)

- from RPi import GPIO (pip install RPi.GPIO) #for Pi4 only

- from matplotlib import pyplot as plt (pip install matplotlib)

- from scipy (pip install scipy)

- import gpiod (pip install gpiod==1.5.4) #for Pi5 only

**Available libraries as example** for Python 3.11. 2

- matplotlib 3.6.3

- spidev 3.5

- RPi.GPIO 0.7.1a4

- scipy 1.10.1
