# Cap

Can use any comfortable caps for you, or create your own cap (DIY cap). Here you can find step by step ***[instructions](https://www.researchgate.net/publication/369378602_DIY_-Hat_for_measuring_EEG_with_dry_electrodes_for_PiEEG) ***how to do that.

We usually use [Cap](https://pieeg.com/cap-eeg-kit/) made of nylon material. All PiEEG devices have a connection Connector Header Through Hole - 2.54mm, so if you use some other caps you need confirm that connctors  has the same view

![Image](https://pieeg.com/wp-content/uploads/2025/05/app-1-2.png)

to connect the Header in PiEEG devices, that have the next pins

![Image](https://pieeg.com/wp-content/uploads/2025/05/app-1-1.png)
