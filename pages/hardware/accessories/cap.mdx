# Cap
### Ready-Made Cap
PiEEG recommends the [Nylon EEG Cap](https://pieeg.com/cap-eeg-kit/).  It is made from a comfortable nylon material and comes pre-configured for use with all PiEEG devices.

### DIY Cap
Can use any comfortable caps for you, or create your own cap (DIY cap). Here you can find step-by-step ***[instructions](https://www.researchgate.net/publication/369378602_DIY_-Hat_for_measuring_EEG_with_dry_electrodes_for_PiEEG)*** on how to do that.


All PiEEG devices have a connection Connector Header Through Hole - 2.54 mm, so if you use some other caps, you need to confirm that the connectors have the same view

### Electrode Placement
Whether using dry or wet electrodes, place them according to the international 10-20 system — the standard method for consistent, reproducible EEG electrode positioning across sessions and subjects.

![Image](/images/hardware/2024/08/New-Bitmap-Image-264x300.jpg)

## Choosing the Right Cap

| Cap Type | Setup Required | Compatibility | Best For |
| :--- | :--- | :--- | :--- |
| **PiEEG Nylon Cap** | None | All PiEEG devices | Recommended starting point for most users |
| **Third-Party Cap** | Verify 2.54 mm connector | Must confirm pin pitch | Custom fit, specific channel counts |
| **DIY Cap** | Build from instructions | 2.54 mm pin pitch required | Experimentation, custom layouts, low cost |
